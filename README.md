## 🚗 What is This Project?

> **Traffic Congestion Predictor** is a **Streamlit web app** that compares two road routes across Mumbai — the **Default (Fastest)** route vs the **Optimized (Shortest)** route — and uses **Machine Learning** to predict realistic travel times and congestion levels by factoring in live weather, time of day, and road conditions.

```python
# Two routes. One smart choice.
default_route   = "Fastest path (ORS)"          # ⚡ Speed-focused
optimized_route = "Shortest + congestion-aware"  # 🧠 ML-adjusted

# Winner isn't always the fastest — it's the smartest 🚦
```

## ⚙️ Installation & Setup

### 🔁 Step 1 — Clone the Repo

```bash
git clone https://github.com/Zameer17/traffic-app.git
cd traffic-app
```

### 📦 Step 2 — Install Dependencies

```bash
pip install -r requirements.txt
```

### 🔑 Step 3 — Add Your API Keys

Create a `.env` file in the root folder:

```env
ORS_API_KEY=your_openrouteservice_api_key
WEATHER_API_KEY=your_openweather_api_key
```

> 🔗 Get ORS key → **[openrouteservice.org](https://openrouteservice.org)**
> 🔗 Get Weather key → **[openweathermap.org](https://openweathermap.org/api)**

### 🚀 Step 4 — Run the App

```bash
streamlit run app.py
```

## 🤔 FAQ

<details>
<summary><b>🚗 Why is the default route sometimes faster than the optimized route?</b></summary>
<br>
The <b>default route (fastest)</b> is calculated by ORS purely based on travel time. The <b>optimized route (shortest)</b> reduces congestion and distance but may not always be quicker in absolute time.

Our ML model adjusts for <b>realistic traffic, congestion, and weather</b> → so the optimized route may be more <b>reliable and consistent</b> even if slightly slower on paper.
</details>

<details>
<summary><b>🔑 Where do I get API keys?</b></summary>
<br>
- <b>OpenRouteService</b>: <a href="https://openrouteservice.org">openrouteservice.org</a> — free tier available<br>
- <b>OpenWeather</b>: <a href="https://openweathermap.org/api">openweathermap.org/api</a> — free tier available
</details>

<details>
<summary><b>🌍 Does it work for cities other than Mumbai?</b></summary>
<br>
Currently optimized and trained for <b>Mumbai road data</b>. With a new dataset and retraining, it can be extended to any city!
</details>
